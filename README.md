# Simulador de Cola en Java

Proyecto de Estructuras de Datos - Universidad Da Vinci de Guatemala  
Curso: Estructuras de Datos  
Cátedro: Ing. Brandon Chitay  

## 📌 Descripción
Este proyecto implementa un simulador de atención al cliente en una sucursal bancaria, utilizando **estructuras de datos Cola y Pila** construidas desde cero en Java.

- **Cola (FIFO)**: gestiona clientes en orden de llegada.  
- **Pila (LIFO)**: guarda el historial de clientes atendidos.
